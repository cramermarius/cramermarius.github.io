## rofi-menus

A small collection of rofi scripts to manage some basic utilities like the following:
* shutting down
* screen brightness
* some xrandr settings
* scrot functions
* i3 layouts

<br/>

~[](/images/rofi-menus.png)

<br/>

Go check it out on [GitHub!](https://github.com/cramermarius/rofi-menus)
